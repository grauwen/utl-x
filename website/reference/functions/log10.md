# log10

<p class="stdlib-meta"><code>log10(number) → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Base-10 logarithm.

- `number` (required): positive number

``` utlx
log10(1000)                              // 3
log10(1)                                 // 0
```
