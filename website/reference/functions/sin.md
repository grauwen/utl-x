# sin

<p class="stdlib-meta"><code>sin(x) → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Compute the sine of an angle in radians.

- `x` (required): angle in radians

``` utlx
sin(pi() / 2)                            // 1.0
```
