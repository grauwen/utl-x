# lowerCase

<p class="stdlib-meta"><code>lowerCase(string) → string</code> · <a href="/reference/stdlib#string">String</a></p>

Convert a string to all lowercase.

- `string` (required): the string to convert

``` utlx
{
  result: lowerCase("Hello World")           // "hello world"
}
```

Also: `camelCase`, `snakeCase`, `kebabCase`, `pascalCase`, `titleCase`,
`dotCase`, `pathCase`, `constantCase`, `slugify`.
