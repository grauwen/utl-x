# toRadians

<p class="stdlib-meta"><code>toRadians(degrees) → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Convert degrees to radians.

- `degrees` (required): angle in degrees

``` utlx
toRadians(180)                           // ~3.14159 (pi)
toRadians(90)                            // ~1.5708
```
