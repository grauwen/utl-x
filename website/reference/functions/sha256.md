# sha256

<p class="stdlib-meta"><code>sha256(data) → string</code> · <a href="/reference/stdlib#security">Security</a></p>

Compute a SHA-256 hash. Returns hex-encoded digest string. See Chapter
38.

- `data` (required): string to hash

``` utlx
sha256("hello")                          // "2cf24dba5fb0a30e..."

// Use case: content-addressed caching
let contentHash = sha256(renderJson($input))
{...$input, hash: contentHash}
```
