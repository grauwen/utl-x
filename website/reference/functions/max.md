# max

<p class="stdlib-meta"><code>max(array) → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Find the maximum value in a numeric array.

- `array` (required): array of numbers

``` utlx
{
  highest: max([3, 1, 4, 1, 5, 9])          // 9
}
```
