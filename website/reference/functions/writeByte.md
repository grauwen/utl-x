# writeByte

<p class="stdlib-meta"><code>writeByte(value) → binary</code> · <a href="/reference/stdlib#binary">Binary</a></p>

Write a single byte value to binary.

- `value` (required): byte value (0-255)

``` utlx
writeByte(65)                            // binary containing byte 0x41
```
