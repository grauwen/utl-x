# ln

<p class="stdlib-meta"><code>ln(number) → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Natural logarithm (base *e*).

- `number` (required): positive number

``` utlx
ln(1)                                    // 0
ln(e())                                  // 1
ln(10)                                   // ~2.302585
```
