# reverseString

<p class="stdlib-meta"><code>reverseString(string) → string</code> · <a href="/reference/stdlib#string">String</a></p>

Reverse the characters in a string.

- `string` (required): the string to reverse

``` utlx
reverseString("hello")                   // "olleh"
```
