# cosh

<p class="stdlib-meta"><code>cosh(number) → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Hyperbolic cosine.

- `number` (required): input value

``` utlx
cosh(0)        // 1.0
cosh(1)        // 1.5430806348152437
```
