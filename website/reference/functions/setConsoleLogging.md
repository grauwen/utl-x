# setConsoleLogging

<p class="stdlib-meta"><code>setConsoleLogging(enabled) → null</code> · <a href="/reference/stdlib#system">System</a></p>

Enable or disable console logging output.

- `enabled` (required): `true` to enable, `false` to disable

``` utlx
let _ = setConsoleLogging(true)
```
