# random

<p class="stdlib-meta"><code>random() → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Generate a random number between 0.0 (inclusive) and 1.0 (exclusive).

``` utlx
{
  sampleId: floor(random() * 1000)
}
```
