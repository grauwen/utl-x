# toDegrees

<p class="stdlib-meta"><code>toDegrees(radians) → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Convert radians to degrees.

- `radians` (required): angle in radians

``` utlx
toDegrees(pi())                          // 180.0
toDegrees(pi() / 2)                      // 90.0
```
