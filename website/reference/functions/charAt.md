# charAt

<p class="stdlib-meta"><code>charAt(string, index) → string</code> · <a href="/reference/stdlib#string">String</a></p>

Get the character at a specific index (0-based).

- `string` (required): the source string

- `index` (required): position (0-based)

``` utlx
charAt("Hello", 0)    // "H"
charAt("Hello", 4)    // "o"
```
