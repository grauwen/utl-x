# getTimezoneName

<p class="stdlib-meta"><code>getTimezoneName() → string</code> · <a href="/reference/stdlib#date-time">Date & Time</a></p>

Get the timezone name/ID for the current system.

``` utlx
{
  tz: getTimezoneName()                  // "Europe/Amsterdam"
}
```
