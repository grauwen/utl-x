# min

<p class="stdlib-meta"><code>min(array) → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Find the minimum value in a numeric array.

- `array` (required): array of numbers

``` utlx
{
  lowest: min([3, 1, 4, 1, 5, 9])           // 1
}
```
