# sinh

<p class="stdlib-meta"><code>sinh(x) → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Compute the hyperbolic sine.

- `x` (required): a number

``` utlx
sinh(1)                                  // ~1.1752
```
