# sqrt

<p class="stdlib-meta"><code>sqrt(x) → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Compute the square root.

- `x` (required): a non-negative number

``` utlx
sqrt(16)                                 // 4.0
sqrt(2)                                  // ~1.4142
```
