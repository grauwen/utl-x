# tan

<p class="stdlib-meta"><code>tan(x) → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Compute the tangent of an angle in radians.

- `x` (required): angle in radians

``` utlx
tan(pi() / 4)                            // ~1.0
```
