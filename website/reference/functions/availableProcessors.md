# availableProcessors

<p class="stdlib-meta"><code>availableProcessors() → number</code> · <a href="/reference/stdlib#system">System</a></p>

Get the number of available CPU cores/processors on the current system.

``` utlx
{
  cpuCores: availableProcessors()
}
// Output: {"cpuCores": 8}
```
