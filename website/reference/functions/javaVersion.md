# javaVersion

<p class="stdlib-meta"><code>javaVersion() → string</code> · <a href="/reference/stdlib#system">System</a></p>

Get the Java/JVM version string of the runtime.

``` utlx
{
  runtime: javaVersion()              // "21.0.2"
}
```
