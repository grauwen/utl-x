# tanh

<p class="stdlib-meta"><code>tanh(x) → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Compute the hyperbolic tangent.

- `x` (required): a number

``` utlx
tanh(1)                                  // ~0.7616
```
