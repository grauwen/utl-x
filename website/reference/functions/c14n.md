# c14n

<p class="stdlib-meta"><code>c14n(xml) → string</code> · <a href="/reference/stdlib#xml">XML</a></p>

Canonicalize an XML document using W3C C14N (sorted attributes,
normalized whitespace). See Chapter 22 for full details.

- `xml` (required): XML UDM value to canonicalize

``` bash
echo '<Order id="1" status="new"/>' | utlx -f xml -e 'c14n($input)'
# canonical XML string (sorted attributes, normalized whitespace)
```

``` utlx
{
  canonical: c14n($input),
  fingerprint: c14nFingerprint($input)
}
```

Also: `c14nWithComments(xml)`, `excC14n(xml)` (exclusive, for SOAP),
`c14n11(xml)` (version 1.1), `c14nFingerprint(xml)` (short hash for
logging).
