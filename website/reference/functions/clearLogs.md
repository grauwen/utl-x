# clearLogs

<p class="stdlib-meta"><code>clearLogs() → null</code> · <a href="/reference/stdlib#system">System</a></p>

Clear all accumulated log entries from the log buffer.

``` utlx
clearLogs()
// all previous log entries are discarded
```
