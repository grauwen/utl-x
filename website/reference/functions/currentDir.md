# currentDir

<p class="stdlib-meta"><code>currentDir() → string</code> · <a href="/reference/stdlib#system">System</a></p>

Get the current working directory path.

``` utlx
{
  workingDir: currentDir()
}
// Output: {"workingDir": "/home/user/project"}
```
