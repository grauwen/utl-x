# log2

<p class="stdlib-meta"><code>log2(number) → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Base-2 logarithm.

- `number` (required): positive number

``` utlx
log2(8)                                  // 3
log2(1024)                               // 10
```
