# upper

<p class="stdlib-meta"><code>upper(string) → string</code> · <a href="/reference/stdlib#string">String</a></p>

Convert a string to uppercase. Legacy naming — prefer `upperCase`.

- `string` (required): the string to convert

``` utlx
upper("hello")                           // "HELLO"
```
