# e

<p class="stdlib-meta"><code>e() → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Returns Euler's number (approximately 2.71828182845904523536).

``` utlx
e()                                      // 2.718281828459045
{
  euler: e(),
  expGrowth: e() ** $input.rate
}
```
