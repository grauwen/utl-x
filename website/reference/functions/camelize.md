# camelize

<p class="stdlib-meta"><code>camelize(string) → string</code> · <a href="/reference/stdlib#string">String</a></p>

Alias for `camelCase()`. Convert a string to camelCase. Prefer
`camelCase()`.

- `string` (required): the string to convert

``` utlx
camelize("my_field_name")       // "myFieldName"
```
