# pow

<p class="stdlib-meta"><code>pow(base, exponent) → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Raise a number to a power.

- `base` (required): the base number

- `exponent` (required): the exponent

``` utlx
pow(2, 10)                               // 1024
pow(10, 3)                               // 1000
```
