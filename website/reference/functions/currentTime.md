# currentTime

<p class="stdlib-meta"><code>currentTime() → time</code> · <a href="/reference/stdlib#date-time">Date & Time</a></p>

Return the current time (no date component).

``` utlx
{
  time: currentTime()                        // 14:30:00 (time only, no date)
}
```
