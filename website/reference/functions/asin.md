# asin

<p class="stdlib-meta"><code>asin(number) → number</code> · <a href="/reference/stdlib#math">Math</a></p>

Arc sine (inverse sine). Returns angle in radians.

- `number` (required): value between -1 and 1

``` utlx
asin(0)        // 0.0
asin(1)        // 1.5707963267948966 (π/2)
asin(-1)       // -1.5707963267948966 (-π/2)
```
